# Agentic AI Chat Starter (Login + Dashboard)

Basic setup for an agentic AI chat app with:

- session-based login/logout
- protected dashboard route
- chat UI + `/api/chat` endpoint
- tiny built-in tool-routing agent (calculator + clock)

## Run

```bash
python3 app.py
```

Open http://localhost:5000
